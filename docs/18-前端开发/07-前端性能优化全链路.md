### 1.1 Core Web Vitals

| 指标                                  | 描述       | 良好     | 需改进    | 差      |

| **FCP** (First Contentful Paint)    | 首次内容绘制   | ≤1.8s  | ≤3s    | >3s    |
| **TTFB** (Time to First Byte)       | 首字节时间    | ≤600ms | ≤800ms | >800ms |
| **INP** (Interaction to Next Paint) | 交互到下一次绘制 | ≤200ms | ≤500ms | >500ms |

### 3.1 虚拟列表

```vue
<!-- 虚拟列表组件 -->
<template>
  <div ref="container" class="virtual-list" @scroll="handleScroll">
    <div class="virtual-list-phantom" :style="{ height: totalHeight + 'px' }"></div>
    <div class="virtual-list-content" :style="{ transform: `translateY(${offset}px)` }">
      <div
          v-for="item in visibleData"
          :key="item.id"
          class="virtual-list-item"
          :style="{ height: itemHeight + 'px' }"
      >
        {{ item.name }}
      </div>
    </div>
  </div>
</template>

<script setup>
  import {ref, computed, onMounted} from 'vue';

  const props = defineProps({
    data: Array,
    itemHeight: {type: Number, default: 50},
    bufferSize: {type: Number, default: 5} // 缓冲区大小
  });

  const container = ref();
  const scrollTop = ref(0);
  const containerHeight = ref(0);

  // 可视区域能显示的数量
  const visibleCount = computed(() =>
      Math.ceil(containerHeight.value / props.itemHeight)
  );

  // 起始索引（带缓冲）
  const startIndex = computed(() =>
      Math.max(0, Math.floor(scrollTop.value / props.itemHeight) - props.bufferSize)
  );

  // 结束索引（带缓冲）
  const endIndex = computed(() =>
      Math.min(
          props.data.length,
          startIndex.value + visibleCount.value + props.bufferSize * 2
      )
  );

  // 可视数据
  const visibleData = computed(() =>
      props.data.slice(startIndex.value, endIndex.value)
  );

  // 总高度
  const totalHeight = computed(() => props.data.length * props.itemHeight);

  // 偏移量
  const offset = computed(() => startIndex.value * props.itemHeight);

  const handleScroll = () => {
    scrollTop.value = container.value.scrollTop;
  };

  onMounted(() => {
    containerHeight.value = container.value.clientHeight;
  });
</script>
```

### 3.3 防抖与节流

```javascript
// 防抖（debounce）- 延迟执行，只执行最后一次
function debounce(fn, delay) {
    let timer = null;
    return function (...args) {
        if (timer) clearTimeout(timer);
        timer = setTimeout(() => {
            fn.apply(this, args);
        }, delay);
    };
}

// 使用场景：搜索输入
const handleSearch = debounce((keyword) => {
    fetchSearchResults(keyword);
}, 300);

// 节流（throttle）- 固定频率执行
function throttle(fn, limit) {
    let inThrottle;
    return function (...args) {
        if (!inThrottle) {
            fn.apply(this, args);
            inThrottle = true;
            setTimeout(() => inThrottle = false, limit);
        }
    };
}

// 使用场景：滚动监听
const handleScroll = throttle(() => {
    console.log('scroll position:', window.scrollY);
}, 100);

// Vue 中使用
import {debounce, throttle} from 'lodash-es';

// 或自定义 Hook
function useDebounce(value, delay) {
    const debouncedValue = ref(value);
    let timer;

    watch(value, (newVal) => {
        clearTimeout(timer);
        timer = setTimeout(() => {
            debouncedValue.value = newVal;
        }, delay);
    });

    return debouncedValue;
}
```

## 六、性能监控与告警

```javascript
// 性能监控 SDK
class PerformanceMonitor {
    constructor(options = {}) {
        this.reportUrl = options.reportUrl;
        this.sampleRate = options.sampleRate || 1;
        this.init();
    }

    init() {
        // 核心指标
        this.observeWebVitals();

        // 资源加载
        this.observeResources();

        // 长任务
        this.observeLongTasks();

        // 错误监控
        this.observeErrors();
    }

    observeWebVitals() {
        import('web-vitals').then(({getCLS, getFID, getFCP, getLCP, getINP}) => {
            getCLS(this.reportMetric.bind(this, 'CLS'));
            getFID(this.reportMetric.bind(this, 'FID'));
            getFCP(this.reportMetric.bind(this, 'FCP'));
            getLCP(this.reportMetric.bind(this, 'LCP'));
            getINP(this.reportMetric.bind(this, 'INP'));
        });
    }

    observeResources() {
        const observer = new PerformanceObserver((list) => {
            for (const entry of list.getEntries()) {
                if (entry.duration > 1000) { // 慢资源
                    this.report({
                        type: 'slow-resource',
                        name: entry.name,
                        duration: entry.duration,
                        initiatorType: entry.initiatorType
                    });
                }
            }
        });
        observer.observe({entryTypes: ['resource']});
    }

    observeLongTasks() {
        const observer = new PerformanceObserver((list) => {
            for (const entry of list.getEntries()) {
                this.report({
                    type: 'long-task',
                    duration: entry.duration,
                    startTime: entry.startTime
                });
            }
        });
        observer.observe({entryTypes: ['longtask']});
    }

    observeErrors() {
        window.addEventListener('error', (e) => {
            this.report({
                type: 'error',
                message: e.message,
                filename: e.filename,
                lineno: e.lineno,
                colno: e.colno
            });
        });

        window.addEventListener('unhandledrejection', (e) => {
            this.report({
                type: 'unhandledrejection',
                reason: e.reason
            });
        });
    }

    reportMetric(name, metric) {
        this.report({
            type: 'web-vital',
            name,
            value: metric.value,
            rating: metric.rating, // 'good' | 'needs-improvement' | 'poor'
            delta: metric.delta
        });
    }

    report(data) {
        if (Math.random() > this.sampleRate) return;

        const payload = {
            ...data,
            url: window.location.href,
            timestamp: Date.now(),
            userAgent: navigator.userAgent,
            // 可以添加用户标识等
        };

        // 使用 sendBeacon 确保数据发送
        if (navigator.sendBeacon) {
            navigator.sendBeacon(this.reportUrl, JSON.stringify(payload));
        } else {
            fetch(this.reportUrl, {
                method: 'POST',
                body: JSON.stringify(payload),
                keepalive: true
            });
        }
    }
}

// 使用
const monitor = new PerformanceMonitor({
    reportUrl: '/api/metrics',
    sampleRate: 0.1 // 10% 采样
});
```

**问题 2：虚拟列表的实现原理是什么？**

**答：**

虚拟列表只渲染可视区域的内容，通过以下方式实现：

1. **计算可视区域**：根据容器高度和 item 高度计算可显示数量
2. **动态截取数据**：只渲染可视区域 + 缓冲区的数据
3. **位移模拟滚动**：使用 transform 偏移模拟完整列表高度
4. **复用 DOM 节点**：滚动时更新数据而非创建新节点

```
总高度 = 数据条数 × 单项高度
可视数量 = Math.ceil(容器高度 / 单项高度)
起始索引 = Math.floor(滚动位置 / 单项高度)
偏移量 = 起始索引 × 单项高度
```

**问题 3：如何检测和解决内存泄漏？**

**答：**

**检测方法：**

1. Chrome DevTools Memory 面板 - Heap Snapshot 对比
2. Performance 面板 - 查看内存增长趋势
3. `performance.memory` API 监控

**常见泄漏场景：**

- 未清理的事件监听、定时器
- 闭包引用大对象
- 全局变量缓存未释放
- DOM 引用未清理

**解决方案：**

- 组件销毁时清理副作用
- 使用 WeakMap/WeakSet
- 避免全局缓存大数据
- 使用虚拟滚动处理长列表

**问题 4：Service Worker 的缓存策略有哪些？**

**答：**

| 策略                         | 说明          | 适用场景        |
|----------------------------|-------------|-------------|
| **Cache First**            | 先读缓存，无缓存再网络 | 静态资源、图片     |
| **Network First**          | 先网络，失败再缓存   | API 请求、实时数据 |
| **Stale While Revalidate** | 返回缓存，同时更新   | 首页、非关键数据    |
| **Network Only**           | 只走网络        | 实时性要求高的请求   |
| **Cache Only**             | 只读缓存        | 离线应用        |

**问题 5：什么是 Time to Interactive (TTI)？如何优化？**

**答：**

**TTI** 指页面可交互的时间，受以下因素影响：

1. **JavaScript 执行** - 长任务阻塞主线程
2. **主线程空闲** - 需要足够的空闲时间

**优化方法：**

- 代码分割，减少首屏 JS 体积
- 延迟加载非关键脚本
- 使用 Web Worker 处理大数据
- 优化长任务，拆分成小任务
- 使用 `requestIdleCallback` 执行低优先级任务
