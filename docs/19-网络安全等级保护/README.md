## 🎯 学习建议

### 等级保护基础概念（1 天）

1. **等保制度背景**
    - 《网络安全法》要求
    - GB/T 22239-2019 标准
    - 等保2.0核心变化

2. **等级划分**
    - 一级：自主保护
    - 二级：指导保护
    - 三级：监督保护
    - 四级：强制保护
    - 五级：专控保护

### 二级等保实践（2 天）

1. **定级备案**
    - 系统识别与定级
    - 备案材料准备
    - 公安机关备案流程

2. **安全建设**
    - 物理安全措施
    - 网络安全加固
    - 主机安全配置
    - 应用安全开发

3. **测评整改**
    - 测评机构选择
    - 测评流程配合
    - 问题整改方案

### 三级等保实践（3 天）

1. **增强安全要求**
    - 身份鉴别增强
    - 访问控制细化
    - 安全审计完善
    - 入侵防范措施

2. **技术体系**
    - 安全通信网络
    - 安全区域边界
    - 安全计算环境
    - 安全管理中心

3. **管理要求**
    - 安全管理制度
    - 安全管理机构
    - 人员安全管理
    - 系统建设管理

## 🛠️ 常见系统定级参考

| 系统类型 | 建议等级  | 依据            |
|:----:|:-----:|:--------------|
| 企业官网 | 一级/二级 | 信息公开为主，影响范围有限 |
| 电商平台 | 二级/三级 | 涉及用户信息、交易数据   |
| 金融系统 | 三级/四级 | 涉及资金安全、用户敏感信息 |
| 医疗系统 |  三级   | 涉及个人健康信息      |
| 政务系统 | 二级/三级 | 涉及公民信息、政务服务   |
| 教育系统 |  二级   | 涉及学生信息、教务数据   |
