## 🎯 学习建议

### 第一阶段：基础概念（1-2 天）

1. **低代码平台概述**
    - 什么是低代码
    - 核心价值和应用场景
    - 平台架构分层

### 第二阶段：代码生成器（3-4 天）⭐重点

1. **模板引擎**
    - Velocity/Freemarker语法
    - 模板变量替换
    - 条件循环语句

2. **代码生成实战**
    - 数据库元数据解析
    - Entity/Mapper/Service/Controller生成
    - 批量代码生成

### 第三阶段：动态表单引擎（3-4 天）⭐重点

1. **JSON Schema 设计**
    - 表单字段描述
    - 验证规则定义
    - 选项类型处理

2. **前端渲染**
    - Vue/React组件渲染
    - Element UI/Ant Design集成
    - 表单联动逻辑

### 第四阶段：流程引擎（4-5 天）

1. **Activiti 基础**
    - BPMN 2.0 规范
    - 流程定义 XML
    - 流程实例管理

2. **任务管理**
    - 用户任务分配
    - 会签或签实现
    - 流程驳回撤回

### 第五阶段：规则引擎 + 报表引擎（3-4 天）

1. **Drools 规则**
    - 规则文件语法
    - KieContainer 加载
    - 规则执行

2. **报表生成**
    - POI Excel操作
    - Freemarker 模板
    - 定时任务调度

## 🛠️ 实战技巧

### 低代码选型口诀

```
简单 CRUD 用生成器，
复杂表单动态配。
工作流选 Activiti，
规则引擎 Drools 好。
报表 POI 加模板，
低代码平台效率高。
```

### 性能优化建议

```
1. 代码生成：增量生成、避免重复
2. 表单引擎：缓存 Schema、懒加载
3. 流程引擎：异步执行、批量处理
4. 规则引擎：规则缓存、会话复用
5. 报表引擎：流式写入、分块处理
```
