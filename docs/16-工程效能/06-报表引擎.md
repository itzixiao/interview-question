## 二、Excel 报表生成（POI）

### 2.1 销售报表示例

```java
@Service
public class ReportGenerator {
    
    /**
     * 生成销售报表
     */
    public void generateSalesReport(List<SaleData> dataList, String outputPath) {
        try (Workbook workbook = new XSSFWorkbook()) {
            Sheet sheet = workbook.createSheet("销售报表");
            
            // 创建标题行
            Row headerRow = sheet.createRow(0);
            String[] headers = {"日期", "产品", "销售额", "数量", "客户"};
            
            for (int i = 0; i < headers.length; i++) {
                Cell cell = headerRow.createCell(i);
                cell.setCellValue(headers[i]);
                cell.setCellStyle(createHeaderStyle(workbook));
            }
            
            // 填充数据
            int rowNum = 1;
            for (SaleData data : dataList) {
                Row row = sheet.createRow(rowNum++);
                row.createCell(0).setCellValue(data.getDate());
                row.createCell(1).setCellValue(data.getProduct());
                row.createCell(2).setCellValue(data.getAmount());
                row.createCell(3).setCellValue(data.getQuantity());
                row.createCell(4).setCellValue(data.getCustomer());
            }
            
            // 自动调整列宽
            for (int i = 0; i < headers.length; i++) {
                sheet.autoSizeColumn(i);
            }
            
            // 写入文件
            try (FileOutputStream fos = new FileOutputStream(outputPath)) {
                workbook.write(fos);
            }
            
            log.info("报表生成成功：{}", outputPath);
        } catch (IOException e) {
            log.error("报表生成失败", e);
        }
    }
    
    /**
     * 创建标题样式
     */
    private CellStyle createHeaderStyle(Workbook workbook) {
        CellStyle style = workbook.createCellStyle();
        Font font = workbook.createFont();
        font.setBold(true);
        font.setFontHeightInPoints((short) 12);
        style.setFont(font);
        style.setFillForegroundColor(IndexedColors.GREY_25_PERCENT.getIndex());
        style.setFillPattern(FillPatternType.SOLID_FOREGROUND);
        return style;
    }
}
```

### 2.2 大数据量优化

```java
/**
 * 使用 SXSSFWorkbook 处理大数据量
 */
public void generateLargeReport(List<Data> dataList, String outputPath) {
    // 在内存中只保留 100 行数据
    SXSSFWorkbook workbook = new SXSSFWorkbook(100);
    
    try {
        Sheet sheet = workbook.createSheet("大数据报表");
        
        // 流式写入数据
        int rowNum = 0;
        for (int i = 0; i < dataList.size(); i++) {
            Row row = sheet.createRow(rowNum++);
            // 填充数据...
            
            // 每 1000 行刷新一次
            if (i % 1000 == 0) {
                System.out.println("已处理 " + i + " 行");
            }
        }
        
        // 写入文件
        try (FileOutputStream fos = new FileOutputStream(outputPath)) {
            workbook.write(fos);
        }
        
    } finally {
        workbook.dispose(); // 清理临时文件
    }
}
```

```java
@Component
public class ScheduledReportTask {
    
    @Autowired

    @Autowired
    private MailService mailService;
    
    /**
     * 每天凌晨 1 点生成日报
     */
    @Scheduled(cron = "0 0 1 * * ?")
    public void generateDailyReport() {
        log.info("开始生成日报...");
        
        // 1. 查询昨天数据
        LocalDate yesterday = LocalDate.now().minusDays(1);
        List<SaleData> dataList = saleMapper.selectByDate(yesterday);
        
        // 2. 生成报表
        String fileName = "日报_" + yesterday + ".xlsx";

            "manager@company.com",
            "销售日报 - " + yesterday,
```

## 五、报表引擎面试题

### 1. 常见报表工具有哪些？

**参考答案：**

**商业工具：**

- **FineReport（帆软）**：国内流行，功能强大
- **SmartBI**：企业级 BI 平台
- **Tableau**：可视化分析
- **Power BI**：微软出品

**开源工具：**

- **JasperReports**：Java 老牌
- **BIRT**：Eclipse 项目
- **Pentaho**：Kettle 同门

**代码库：**

- **Apache POI**：Excel操作
- **iText**：PDF 生成
- **Freemarker/Velocity**：模板引擎

### 2. POI 生成 Excel 的优化技巧？

**参考答案：**

**性能优化：**

1. **使用 SXSSF**：流式 API，内存中只保留部分数据
2. **避免频繁创建对象**：复用 CellStyle、Font
3. **批量写入**：减少 IO 次数
4. **关闭自动调整列宽**：autoSizeColumn 很耗时

**内存优化：**

```java
// 使用 SXSSFWorkbook
SXSSFWorkbook workbook = new SXSSFWorkbook(100); // 窗口大小 100

// 及时清理临时文件
workbook.dispose();
```

**大数据处理：**

- 分批次写入
- 使用异步处理
- 考虑 CSV 格式替代

### 3. 如何实现大数据量报表导出？

**参考答案：**

**方案一：分页查询 + 流式写入**

```java
int pageSize = 1000;
int pageNum = 1;
while (true) {
    List<Data> page = mapper.selectPage(pageNum, pageSize);
    if (page.isEmpty()) break;
    
    // 流式写入 Excel
    writeToFile(page);
    pageNum++;
}
```

**方案二：异步处理**

1. 用户发起导出请求
2. 后台异步生成报表
3. 生成完成后通知用户下载

**方案三：分卷压缩**

- 超过 100 万行拆分为多个文件
- 打包为 ZIP 下载

**注意事项：**

- 设置超时时间
- 限制最大导出量
- 提供进度查询

### 4. 如何实现定时报表？

**参考答案：**

**实现步骤：**

1. **定时任务调度**：Spring Schedule / Quartz
2. **数据查询**：按时间范围查询
3. **报表生成**：Excel/PDF/HTML
4. **结果分发**：邮件/FTP/消息推送

**调度框架选择：**

- **简单场景**：Spring Schedule（@Scheduled）
- **复杂场景**：Quartz（支持 Cron、集群）
- **分布式**：XXL-Job（支持分片、故障转移）

**最佳实践：**

- 任务幂等性（失败重试不重复）
- 执行监控（日志、告警）
- 数据备份（防止丢失）

### 5. 如何将报表以邮件形式发送？

**参考答案：**

**实现方式：**

1. **附件形式**：生成文件后作为附件
2. **正文形式**：HTML 直接展示在邮件正文
3. **混合形式**：正文预览 + 附件下载

**代码示例：**

```java
@Autowired
private JavaMailSender mailSender;

public void sendReportEmail(String to, String subject, 
                            String content, String attachmentPath) {
    MimeMessage message = mailSender.createMimeMessage();
    
    try {
        MimeMessageHelper helper = new MimeMessageHelper(message, true);
        helper.setTo(to);
        helper.setSubject(subject);
        helper.setText(content, true); // HTML
        
        // 添加附件
        FileSystemResource file = new FileSystemResource(attachmentPath);
        helper.addAttachment("报表.xlsx", file);
        
        mailSender.send(message);
    } catch (MessagingException e) {
        log.error("邮件发送失败", e);
    }
}
```
