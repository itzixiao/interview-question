## 🎯 学习建议

### HashMap（2 天）
1. **底层结构**
   - JDK7 vs JDK8 差异
   - 数组 + 链表 + 红黑树

2. **核心机制**
   - hash 计算与扰动函数
   - 扩容因子与阈值
   - 树化条件

### 树形结构（3 天）
1. **二叉搜索树**
   - 插入、删除、查找

2. **平衡树**
   - AVL 树旋转操作
   - 红黑树性质与插入

3. **多路查找树**
   - B 树（数据库索引）
   - B+ 树（MySQL 索引）

### 常见算法（3-4 天）
1. **排序算法**
   - 快速排序、归并排序、堆排序
   - 时间复杂度分析、稳定性对比

2. **查找算法**
   - 二分查找及其变体
   - 快慢指针技巧

3. **动态规划**
   - 斐波那契数列、爬楼梯
   - 最长上升子序列、0-1 背包
   - 打家劫舍、最小路径和、不同路径

4. **树形算法**
   - 二叉树遍历（递归/迭代）
   - BST 验证、最大深度
   - 树的 DFS/BFS 遍历

5. **经典问题**
   - 两数之和、合并区间
   - 链表操作、字符串处理
   - 回溯法、DFS/BFS
   - 跳跃游戏、滑动窗口
   - 岛屿问题、沉岛思想

## 🛠️ 实战技巧

### HashMap 遍历方式
```java
// 推荐：entrySet 遍历
for (Map.Entry<K, V> entry : map.entrySet()) {
    System.out.println(entry.getKey() + "=" + entry.getValue());
}

// 或使用 forEach
map.forEach((k, v) -> System.out.println(k + "=" + v));
```

### LRU 缓存实现
```java
class LRUCache<K, V> extends LinkedHashMap<K, V> {
    private final int capacity;
    
    public LRUCache(int capacity) {
        super(capacity, 0.75f, true);
        this.capacity = capacity;
    }
    
    @Override
    protected boolean removeEldestEntry(Map.Entry eldest) {
        return size() > capacity;
    }
}
```
