## 🎯 学习建议

### 第一阶段：语法基础（2-3 天）
1. **数据类型与运算符**
   - 8 种基本类型及其包装类
   - 自动装箱与拆箱
   - 字符串拼接性能优化

2. **面向对象核心**
   - 接口与抽象类的本质区别
   - 多态的实现原理
   - 深拷贝与浅拷贝的应用场景

### 第二阶段：高级特性（3-4 天）
1. **泛型机制**
   - 类型擦除原理
   - 通配符 ? 的使用
   - 泛型方法 vs 泛型类

2. **反射与类加载**
   - Class 对象的获取方式
   - 运行时操作类的方法
   - 双亲委派模型及打破

### 第三阶段：面试冲刺（2-3 天）
1. **高频考点**
   - String 的不可变性设计
   - equals() 与 hashCode() 的关系
   - 泛型在集合框架中的应用

2. **实战演练**
   - 手写单例模式（双重检查锁）
   - 自定义类加载器
   - 反射调用私有方法

## 🛠️ 实战技巧

### 使用反射创建对象
```java
Class<?> clazz = Class.forName("com.example.MyClass");
Object obj = clazz.getDeclaredConstructor().newInstance();
```

### 打破双亲委派
```java
class MyClassLoader extends ClassLoader {
    @Override
    protected Class<?> loadClass(String name, boolean resolve) 
        throws ClassNotFoundException {
        // 先自己加载，找不到再委托给父加载器
        return findClass(name);
    }
}
```
