# AI视频生成从零到一实战指南

## 一、AI视频生成概述

### 1.1 什么是AI视频生成

AI视频生成是利用人工智能技术（大模型、扩散模型等），通过**文本描述、图片、脚本**等输入，自动生成视频内容的技术。目前已广泛应用于**短视频创作、小说推文、知识科普、电商带货、影视预告**等场景。

### 1.2 AI视频生成的核心能力

| 能力 | 说明 | 典型工具 |
|:---:|------|---------|
| 文生视频 | 输入文字描述，直接生成视频片段 | 可灵AI、Runway、Pika、Sora |
| 图生视频 | 上传图片，AI生成动态视频 | 可灵AI、Runway Gen-3、Stable Video |
| 文生图 | 输入文字生成图片素材 | Midjourney、DALL·E 3、Stable Diffusion |
| AI配音 | 文字转语音，生成旁白/对话 | 剪映TTS、微软Azure TTS、ElevenLabs |
| AI换脸/数字人 | 生成虚拟主播或角色 | HeyGen、D-ID、腾讯智影 |
| AI字幕 | 自动识别/生成字幕 | 剪映、CapCut、Whisper |
| AI脚本 | 自动生成视频文案/脚本 | ChatGPT、文心一言、通义千问 |

## 三、小说推文视频 — 从0到1全流程

> 小说推文是目前AI视频生成最热门的赛道之一，通过将小说片段制作成短视频，引导用户付费阅读，实现变现。

### 3.1 整体流程

```
选择小说 → 提取爆点片段 → AI生成脚本 → AI生成图片/视频 → AI配音 → 剪辑合成 → 发布推广
```

### 3.2 第一步：选择小说与平台

#### 合作平台（获取推广资格）

| 平台 | 类型 | 佣金模式 | 入驻要求 |
|:---:|------|---------|---------|
| **番茄小说** | 免费小说 | CPA（按付费用户） | 粉丝1000+ |
| **七猫小说** | 免费小说 | CPA | 粉丝500+ |
| **掌阅** | 付费小说 | CPS（按充值分成） | 有作品要求 |
| **书旗小说** | 免费+付费 | CPA/CPS | 粉丝要求较低 |

#### 选书技巧

1. **选爆款题材**：赘婿逆袭、重生复仇、总裁甜宠、玄幻修仙
2. **找高潮片段**：打脸反转、身份揭秘、绝境翻盘
3. **看数据指标**：热度榜单、完读率、推荐量
4. **参考同行**：看抖音/快手上热门的小说推文用的什么书

#### AI选书分析提示词模板

```
# 角色
你是一个小说推文选品专家，深谙抖音/快手爆款内容的底层逻辑。

# 任务
我要做小说推文短视频，请帮我分析以下小说是否适合做推文视频。

# 分析维度（逐项评估，1-10分）
1. 【钩子密度】前10章是否有足够多的反转/冲突/打脸情节？
2. 【情绪张力】能否在30秒内制造强烈的情绪波动（屈辱→反转→爽感）？
3. 【画面感】场景描写是否有强烈的视觉冲击力，适合AI生图？
4. 【角色辨识度】主角是否有鲜明特征，容易让观众记住？
5. 【悬念设置】每个高潮后是否自然留有悬念，引导继续阅读？
6. 【受众匹配】目标受众是否与短视频主力人群（18-35岁）匹配？

# 小说信息
- 书名：{书名}
- 题材：{题材类型}
- 简介：{粘贴小说简介}
- 前几章内容摘要：{粘贴前3-5章关键情节}

# 输出格式
| 维度 | 评分(1-10) | 分析说明 |
最终结论：推荐/不推荐，理由：...
如果推荐，请指出最适合做短视频的3个高潮片段（标注章节位置）。
```

#### AI爆点片段提取提示词模板

> 找到了合适的小说后，用AI快速提取最有"爆点"的片段：

```
# 角色
你是一个短视频内容策划，擅长从长篇小说中精准找到最适合做短视频的高潮片段。

# 任务
请阅读以下小说内容，从中提取5个最适合做30-60秒短视频的"爆点片段"。

# 什么是好的"爆点片段"
1. 有强烈的【情绪反差】：屈辱→逆袭、误会→真相、弱小→强大
2. 有清晰的【视觉画面】：场景具体，动作明确，适合AI生成图片
3. 有天然的【悬念钩子】：片段结束时观众会想"后面怎样了？"
4. 有可传播的【金句/台词】：一句话能概括整个冲突
5. 【独立成段】：不需要大量前情铺垫，30秒内能讲清楚

# 小说内容：
{粘贴小说章节内容}

# 输出格式（每个爆点片段）
## 爆点{编号}：{一句话概括}
- **所在位置**：第X章，第X段
- **核心冲突**：谁 vs 谁，因为什么
- **情绪曲线**：开始情绪 → 转折点 → 结束情绪
- **关键台词**：最有冲击力的一句对白
- **视觉画面**：3个最有画面感的瞬间
- **推荐视频时长**：30秒/60秒
- **爆款潜力评分**：⭐~⭐⭐⭐⭐⭐
```

#### 实操SOP：选书到提取片段（5步完成）

| 步骤 | 操作 | 工具 | 耗时 |
|:---:|------|------|:----:|
| 1 | 打开番茄/七猫热度榜，筛选前20本热门小说 | 番茄小说APP | 10分钟 |

### 3.3 第二步：AI生成脚本

使用ChatGPT、文心一言、通义千问等AI工具生成视频脚本。核心思路：**给AI一个明确的角色 + 具体的约束条件 + 固定的输出格式**。

#### 模板一：通用小说推文脚本（万能模板）

```
# 角色设定
你是一个拥有10年经验的短视频编导，擅长将小说高潮片段改编为爆款短视频脚本。

# 任务描述
请根据以下小说片段，生成一个{30/60/90}秒左右的短视频脚本。

# 硬性要求
1. 【钩子】开头3秒必须有强烈的冲突、反转或悬念，让用户停下来
2. 【节奏】每5-8秒切换一次画面，保持快节奏
3. 【高潮】中间部分展开核心冲突，情绪逐步拉满
4. 【悬念】结尾必须在最高潮处戛然而止，设置"未完待续"的悬念
5. 【画面描述】每个画面需要详细描述（人物外貌、表情、动作、场景、光线），可直接用于AI生图
6. 【旁白】旁白文字口语化，有情绪感，适合配音朗读
7. 【竖屏构图】所有画面按9:16竖屏比例构思

# 小说题材：{赘婿逆袭/重生复仇/总裁甜宠/玄幻修仙/悬疑推理}
# 小说片段：
{粘贴小说内容}

# 输出格式（严格按此表格）
| 序号 | 时间段 | 画面描述（详细） | 旁白文字 | 情绪/音乐 | 运镜方式 |
```

### 3.4 第三步：AI生成图片素材

#### 方案一：Midjourney 生图

**MJ提示词万能公式：**

```
[主体人物/场景] + [动作/表情] + [环境/背景] + [风格修饰] + [光影] + [镜头] + [画质] + [参数]
```

**MJ提示词模板 — 人物场景类：**

```
{人物描述}, {动作/表情}, {场景环境},
{art style}, {lighting}, {camera angle},
ultra detailed, masterpiece, best quality, 8k,
--ar 9:16 --v 6.1 --s {0-1000} --q 2
```

**填充示例：**

| 槽位 | 赘婿逆袭风格 | 甜宠风格 | 玄幻修仙风格 |
|------|------------|---------|------------|
| 人物描述 | A young Chinese man, sharp eyebrows, short black hair, tall and fit | A cute Chinese girl, big eyes, long hair, ponytail | A young Chinese immortal cultivator, white robe, flowing hair |
| 动作/表情 | kneeling on the ground, looking down, rain on face | smiling shyly, holding a letter | floating in the air, eyes glowing blue, hands forming a seal |
| 场景环境 | in front of a luxury mansion gate, rainy night | in a cozy cafe, warm sunlight through window | on a mountain peak above the clouds, ancient Chinese temple |
| 风格 | cinematic, realistic, dramatic | soft anime style, romantic, warm tones | Chinese fantasy art, epic, mystical |

3. **情感**：根据剧情选择不同情感强度
4. **背景音乐**：音量控制在配音的20-30%

## 九、高频面试题（AI视频方向）

### Q1：目前主流的AI视频生成模型有哪些？各有什么特点？

**答：** 主要包括：
- **Sora（OpenAI）**：支持60秒长视频，物理世界理解能力强，画面一致性好
- **可灵AI（快手）**：国内领先，支持文/图生视频，中文生态友好
- **Runway Gen-3**：专业级工具，运镜控制精准，效果稳定
- **Pika**：操作简单，适合快速生成短片段
- **Stable Video Diffusion**：开源方案，可本地部署，灵活性高

### Q2：AI视频生成的技术原理是什么？

**答：** 核心技术路线：
1. **扩散模型（Diffusion Model）**：从噪声逐步去噪生成视频帧
2. **Transformer架构**：处理时序信息，保证帧间一致性
3. **文本编码器（CLIP等）**：将文字描述转化为视频引导信号
4. **VAE（变分自编码器）**：在压缩的潜空间中生成，降低计算成本
5. **时间注意力机制**：确保视频时间维度的连贯性

### Q3：如何保证AI生成视频中角色的一致性？

**答：**
1. 使用固定的角色描述提示词
2. 利用参考图片（图生图/图生视频模式）
3. 训练角色LoRA模型（Stable Diffusion）
4. 使用IP-Adapter技术保持人物特征
5. 后期通过AI换脸工具统一角色外观

### Q4：小说推文视频的核心指标有哪些？

**答：**
- **完播率**：最重要的指标，决定推荐量（目标 > 30%）
- **点赞率**：反映内容吸引力（目标 > 3%）
- **评论率**：反映用户参与度
- **转化率**：点击小说链接的比例（核心变现指标）
- **ROI**：投入产出比（付费推广时关注）

### Q5：AI视频创作中常见的法律风险有哪些？

**答：**
1. **著作权风险**：未授权使用他人小说/图片/音乐
2. **肖像权风险**：AI换脸使用他人肖像
3. **虚假宣传**：夸大内容或诱导消费
4. **深度伪造**：生成误导性内容可能违反法规
5. **平台违规**：违反各平台的内容发布规则

**防范措施**：使用正规授权内容、避免使用真人肖像、遵守平台规则、关注AI相关法规动态。
