## 🎯 学习建议

### 入门体验（1-2 天）

1. **了解AI视频生态**
    - 主流AI视频生成工具（可灵AI、Runway、Pika、Sora）
    - 文生视频、图生视频、数字人等能力
    - 国内外工具优劣对比

2. **动手体验**
    - 注册可灵AI、剪映
    - 完成第一次文生图、图生视频
    - 用剪映剪出第一个完整视频

### 小说推文实战（3-5 天）

1. **选书与脚本**
    - 注册番茄/七猫推广平台
    - AI生成视频脚本（提示词技巧）
    - 选择爆款题材和高潮片段

2. **素材制作**
    - AI生图（角色一致性技巧）
    - 图生视频（运镜提示词）
    - AI配音（音色选择与语速控制）

3. **发布运营**
    - 标题与封面优化
    - 发布时间选择
    - 数据分析与复盘

### 进阶提升（1-2 周）

1. **品质提升**
    - Midjourney / Stable Diffusion 进阶
    - 高级剪辑技巧
    - 多音色配音与SSML

2. **批量化运营**
    - 标准化SOP建立
    - 多账号矩阵策略
    - 自动化工作流
